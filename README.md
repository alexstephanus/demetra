# Demetra
Demetra is a low-cost, local-first hydroponic controller.  It manages your system's pH, nutrient, and ORP levels, runs fertigation cycles, and can keep you reservoir topped up from an external water line or tank.

It reads the following set of sensors:
 - pH
 - ORP
 - Conductivity/TDS
 - Temperature
