# demetra
Demetra is a low-cost, local-first hydroponic controller.  It manages your system's pH, nutrient, and ORP levels, runs fertigation cycles, and can keep you reservoir topped up from an external water line or tank.

You should not build it right now.  As of this writing (3/23/2026), I'm working on reducing sensor noise for better accuracy and faster readings. I think I've got a bead on the issue, but it'll take at least a few weeks to get a test board fabricated, confirm the change works, and integrate those changes into the motherboard.  If you build it right now I can't guarantee the existing hardware will be compatible with new firmware.


It's built using off-the-shelf components, and designed end-to-end for ease of assembly and operation.  You can view assembly docs at [alexstephanus.github.io/demetra](https://alexstephanus.github.io/demetra).  Again, I strongly recommend against building it until the above-mentioned sensor changes are in place.
